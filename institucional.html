Quem Somos

Thukera Music Store, é uma loja virtual ficticia voltada para vendas de instrumentos musicais, acessórios e utilitários
Esta Página E-Commerce foi desenvolvida à fins academicos, como trabalho de conclusão do curso de "Analise e Desenvolvimento de Sistemas", ministrado no Centro Universitario Senac Santo Amaro.

Sistema

A página foi desenvolvida em HTML5, CSS e JavaScript. Utilizei para o FrontEnd os Frameworks Bootstrap 4.2.1, Swiper 4.x e Jquery.
Esta pagina está hospedada no GitHub de forma estática, como apresentação de Portifólio, mas este site possuí um Backend, desenvolvido inteiramente em Java
Utilizando os frameworks Maven, Spring Data JPA, Hybernate e Thymeleaf
Parcialmente funcional, permite ao usuário se cadastrar através do site. O sistema armazena e gerencia informações dos usuários, dos produtos e dos registros de vendas.
